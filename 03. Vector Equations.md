# Vector Equations
## Vector
#### 개념
* vector : 하나의 column만을 가진 행렬이다.
* R<sup>n</sup> : 모든 n*1 벡터의 집합이다.
* 두 벡터가 같은 크기를 가지고 모든 원소의 값이 같으면 equal하다고 한다.
